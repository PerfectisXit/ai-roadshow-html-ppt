# AI Roadshow HTML PPT

单文件 HTML 路演版 PPT 项目。

## 项目内容

- `ai-roadshow.html`
  - 主实现文件
- `路演思路.md`
  - 原始内容源
- `README_AI接手指南.md`
  - 面向 AI 接手和继续修改的总指导文档
- `ai-roadshow-s21-master.html`
  - 正文页视觉母版参考
- `ai-roadshow-chapter-master.html`
  - 章节页视觉母版参考

## 特点

- 单文件 HTML 实现
- 固定 `16:9` 演示画幅
- 键盘翻页
- 大量内嵌动画、拟真组件和类产品界面演示
- 统一暖白母版风格

## 使用方式

直接用浏览器打开：

```bash
open ai-roadshow.html
```

或双击本地文件打开。

## 说明

这个仓库的核心产物是本地演示文件，不依赖后端服务。  
后续继续精修时，优先阅读 [README_AI接手指南.md](./README_AI接手指南.md)。
