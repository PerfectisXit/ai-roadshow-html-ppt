# 增强版对话组件（Enhanced playChatDemo）使用参考

> H-01 产出物说明。增强版已写入 `ai-roadshow.html`，向后兼容。

## 消息类型速查

| type | 用途 | 谁发的 | 核心效果 |
|------|------|--------|---------|
| `text` | 普通文字（默认） | user / ai | 打字机流式输出（原有行为不变） |
| `file-upload` | 用户上传文件 | user | 文件卡片从右侧滑入 |
| `file-output` | AI 生成/返回文件 | ai | 进度条 → 完成 ✓ 弹出 |
| `image-upload` | 用户发送图片 | user | 图片占位框从右侧滑入 |
| `image-output` | AI 生成图片 | ai | "正在生成" → 模糊渐清 |
| `code` | AI 输出代码块 | ai | 深色代码块逐行打字 |
| `chart` | AI 生成柱状图 | ai | 柱子从零长出 + 数值淡入 |
| `table` | AI 输出数据表格 | ai | 表格逐行淡入 + 异常标红 |

## JSON 格式参考

### text（默认，无需写 type）
```json
{"role":"user", "text":"翻译成中文，保留原文分段结构。"}
{"role":"ai", "text":"已完成翻译，共 20 段。"}
```

### file-upload
```json
{
  "role": "user",
  "type": "file-upload",
  "fileName": "英文行业报告.pdf",
  "fileSize": "20 页",
  "fileIcon": "pdf"
}
```
`fileIcon` 可选值：`pdf`（红）、`docx`（蓝）、`xlsx`（绿）、`pptx`（橙）

### file-output
```json
{
  "role": "ai",
  "type": "file-output",
  "text": "已生成简报初稿：",
  "fileName": "班子简报_初稿.docx",
  "fileSize": "780 字",
  "fileIcon": "docx"
}
```
动画：先显示思考中 → 进度条从 0→100% → 绿色 ✓ 弹出

### image-upload
```json
{
  "role": "user",
  "type": "image-upload",
  "imageAlt": "合锦观澜项目现场照片"
}
```

### image-output
```json
{
  "role": "ai",
  "type": "image-output",
  "text": "已生成配图：",
  "imageAlt": "锦度长租公寓外景效果图"
}
```
动画：先显示"🎨 正在生成图片…" → 模糊渐清

### code
```json
{
  "role": "ai",
  "type": "code",
  "text": "好的，代码如下：",
  "code": "<div class=\"chat-demo\">\n  <div class=\"bubble user\">你好</div>\n</div>",
  "codeLang": "html"
}
```
动画：代码逐行出现（每行约 50ms）

### chart
```json
{
  "role": "ai",
  "type": "chart",
  "text": "柱状图已生成：",
  "chartData": {
    "labels": ["锦方", "锦合", "锦度", "锦兰荟", "锦致", "合锦观澜"],
    "values": [108, 114, 84, 100, 102, 47],
    "colors": ["good", "good", "warn", "good", "good", "warn"]
  }
}
```
`colors` 可选值：`good`（cyan 蓝）、`warn`（red 红）
动画：柱子从 0 高度生长 → 数值淡入

### table
```json
{
  "role": "ai",
  "type": "table",
  "text": "汇总完成，低于 80% 已标红：",
  "tableData": [
    ["项目", "利润完成率", "销售完成率", "回款完成率"],
    ["锦方", "108%", "62% ⚠", "27% 🔴"],
    ["锦合", "114%", "65% 🔴", "55% 🔴"],
    ["锦度", "84%", "70% 🔴", "55% 🔴"]
  ]
}
```
含 `🔴` 或 `⚠` 的单元格自动标红。表格逐行淡入（每行 180ms）。
