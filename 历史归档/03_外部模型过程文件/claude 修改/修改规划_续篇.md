### P45 推进路径 [原s50 重大改造]
- **修改程度**：★ 重大改造
- 标题改为："从今天开始——各部门梳理可 AI 化的工作流"
- 讲演目标：不是列规划，是给行动入口
- 核心思路改为**三层渐进路径**：

  **第一层（最简单）：一句话小工具**

  - 不需要编程基础，对话几轮就能做出来
  - 标签：个人级 | 当天可完成

  **第二层（中等）：固定化 Agent 工作流**
  - 把重复性强的工作流程设计成 Agent 可执行的标准流程
  - 比如"每月经营分析汇总"、"每周竞品数据搜集"、"合同条款风险初筛"
  - 标签：部门级 | 1-2 周可搭建

  **第三层（进阶）：公司级数据平台**
  - 将知识库、经营数据、制度文件纳入统一的 AI 可检索系统
  - 标签：公司级 | 3-6 个月逐步建设

- 视觉重心：**渐进式阶梯组件**
  - 三层台阶，从矮到高，每层有图标 + 标题 + 一句话 + 标签
  - 动画：从左到右逐层亮起

// 柱状图生长动画
async function animateChartGrow(bubble, chartData) {
  const chart = bubble.querySelector(".mini-bar-chart");
  if (!chart) return;
  const bars = chart.querySelectorAll(".bar-fill");
  const values = chart.querySelectorAll(".bar-value");
  const maxVal = Math.max(...chartData.values) * 1.1;
  
  // 延迟后让柱子开始生长
  await sleep(300);
  bars.forEach((bar, i) => {
    const targetHeight = (chartData.values[i] / maxVal) * 80; // 80px 最大高度
    bar.style.height = targetHeight + "px";
  });
  
  // 柱子长完后显示数值
  await sleep(1200);
  values.forEach(v => v.classList.add("show"));
}

  let rightDoneFlag = false;

  const playLeft = async () => {

    rightStat.classList.add("show");

  await Promise.all([playLeft(), playRight()]);

  // 左侧每条 1200-1600ms × 8条 = 约 11 秒

  // 差异约 7 秒，对比效果非常强烈
